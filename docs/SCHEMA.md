
# Schema

## `type`

- `number`
  - int
  - float
- `datetime`
  - date
  - datetime
- `text`
  - random text generated by regexp
  - random sample of provided text
  - structural text
    - `zipcode`, `state`, ...

## `range`

- `number`
- `datetime`
- `misc`

## `format`

- `sprintf` formatter
- `datetime` formatter
- `callback` formatter

## `dependency`

- `sex` -> `firstname`
- `uid` -> `email`
- `sid` -> `dept_id` -> `dept_name`

## `domain`

- `address`
  - zipcode
  - state
  - county
  - city
  - street
  - block
  - building
  - no
- `name`
  - firstname
    - *male*
    - *female*
  - lastname
- `tel`
  - area
  - city
  - pbx
  - ext
- `isbn`
- `email`
